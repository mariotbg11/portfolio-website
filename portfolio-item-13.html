Project 13 Loopstudio Landing Page

Personal Project

The third landing page project and the most recent challenge I took from Frontend Mentor. This challenge is to create a Loopstudios landing page

This challenge is my favorite challenge that I have done so far from Frontend Mentor, because it has a simple and modern design, I built this website using semantic HTML and styling using CSS custom properties. The most difficult part of this challenge is how to properly position the text element above the image element.

I also made this project responsive for mobile devices and created a navigation toogle menu using Javascript that can function when users click on the menu icon. I am very grateful to find a site like Frontend Mentor which is very helpful in improving my frontend skills by building real projects because there are many developers out there who are learning and getting stuck because they don't have an idea of what projects they should build.
