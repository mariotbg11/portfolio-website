Project 04 Omnifood

Course Project

After my scrimba pro account expired I moved on to other courses, I found an HTML & CSS course on Udemy that was on sale and had good reviews. The reason why I returned to learning HTML & CSS was because I felt that the skills I had were still not enough and wanted to deepen them even more by directly building a more advanced project. So I decided to buy this course called Build Responsive Real-World Websites With HTML and CSS by Jonas Schmedtmann.

In this course I learned many things starting from HTML fundamentals such as learning how to structure a good HTML document, using semantic HTML, text elements, image elements, hypelinks and many more. After that is learning the fundamentals of CSS, in my opinion this part is the most important to learn from this course because CSS has a lot of properties that are very useful in building a website, starting from learning text styling, knowing the various selectors that can be used in CSS, how to choose the right font/typography and colors on the website, then getting to know the Flexbox and CSS Grid properties to build/set the layout on a website.

After learning the basics of HTML & CSS, at the end of this course there will be a final project to build a website called Omnifood, a technology company with a main focus on consumer well-being through healthy diets. In creating this website, I followed each step through the video, applying what I i've learned before. This website has five main sections such as hero, how it works, plans, testimonials, and call to action/forms, as well as header and footer. Then make the website responsive depending on the user's device such as laptop, tablet or mobile using media queries.
