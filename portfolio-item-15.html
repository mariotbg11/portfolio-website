Project 15 Multipage Coffee Website

Personal Project

This is multipage website project that i build from scratch, i completed this project for submission on Progate Indonesia Frontend Developer Course learning path.

I built this project by slicing the figma design into a website using HTML & CSS, this website is a coffee and tea brand that sells their products. There are several sections such as store profile, menu products, store location and also contact section. In this project I also learned how to create an accordion and css animation scroll using Javascript DOM technique. I completed this project in about 10 days.
