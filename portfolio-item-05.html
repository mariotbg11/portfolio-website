Project 05 Frontend Dev Portfolio

Personal Project

After completing a course on Udemy and learning a lot of HTML & CSS as well as web design, I wanted to create a landing page website with my own ideas from scratch.

I decided to build a simple portfolio website, I followed the steps as I had learned in the previous course, first I made a wireframe to set the layout and content of the website that I wanted to build and then I chose the type of font and the main color that would be characteristic of the website.

There are five main sections that will be displayed from this website such as hero, about, projects, skills, and call to action/forms. This website is also made responsive so that it can be accessed on various devices such as laptops, tablets and mobile phones. Before deploying this website I tested the quality of the webstie using Lighthouse chrome devtool which is an open-source, automated tool for improving the quality of web pages.
