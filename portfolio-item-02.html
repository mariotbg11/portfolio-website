Project 02 Basketball Scoreboard

Personal Project

In Module 3 of Scrimba Frontend Developer Career Path, the main focus is on making websites interactive using the Javascript programming language.

In javascript challenge part 1 in this module, there is a solo project to create a Basketball Scoreboard where there is a button function to display a number or score every time the button is clicked. There are three button functions with the sum of +1, +2, and +3 just like the score system in a basketball match.
