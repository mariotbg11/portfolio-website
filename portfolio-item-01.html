Project 01 Hometown Homepage

Personal Project

This is my first HTML & CSS solo project, back in October last year I got a coupon for a free month of Scrimba Pro. Scrimba is a platform to learn coding, there are many kinds of course to learn but not all of them can be accessed for free so you need a Scrimba Pro account. One of them is Frontend Developer Career Path

The goal of this project is to build a simple one-page website, by providing design examples from figma to guide in the process of creating web pages so that the results can be the same as the design. In this project flexbox is used to organize the page layout, but the page is not responsive yet.
